# ZiQing（紫青）· 英语口语闯关宇宙

学英语：从幼儿园到雅思托福。面向新手的 Windows 桌面口语学习应用。

## 产品定位

不是冷冰冰的刷题软件，而是 **口语冒险家** 成长产品：

- 冒险地图：咿呀村 → … → 托福城 / 雅思塔  
- 今日副本、随机弹窗短练  
- 待翻盘（错题本）、荣誉墙、复仇墙  
- 答对礼花、答错巴掌等商业化动效  

## 技术栈（已定）

- Electron + React + TypeScript + Vite  
- 本地存储；系统托盘 + 通知弹题
